== Description ==
A clean and simple plugin to automatically add a beautiful shadow effect to all the images in your blog posts.

Simply activate the plugin and see the magic. No configuration needed!

Adding a box shadow around your blog post images will make your articles look more professional and well designed.

There's no extra unnecessary code in the plugin, thus you can stay assured that it won't affect your site's performance at all.

== Installation ==
1. Upload the folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the “Plugins” menu in WordPress.
